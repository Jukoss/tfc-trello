# From client folder
### Install
```
$ npm install
```

## Run
```
$ npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

# From hasura folder

### Install
```
$ npm install
```
```
$ docker compose up
```

## Run
```
$ npm run start
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
